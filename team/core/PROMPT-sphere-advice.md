# Промт: Генерация AI-совета по балансу сфер

## Когда запускать
Каждый раз когда обновляются задачи в `partners/tasks.json` или вручную по запросу.

## Что делать

1. Прочитай `partners/tasks.json` — список задач проекта
2. Прочитай `team/core/analysis.json` — состояние команды
3. Прочитай `network/data.json` — сеть участников (продажи, предоплаты)

## Сферы проекта (7 штук)

| Сфера | Что входит | Вес значимости |
|-------|-----------|----------------|
| Амбассадоры | Работа с амбассадорами, продажи нейробендов, реферальная сеть, обучение амбассадоров | Обычный |
| IT | Разработка бота, сайт, интеграции API, автоматизация, код | Обычный |
| Методология | Протоколы MNPI, дизайн исследования, научная часть, PI-индекс | Обычный |
| Партнёрства | Клиники, партнёры, Neiry, LQ Lab, внешние контакты | Обычный |
| Маркетинг | Контент, статьи, презентации, соцсети, бренд | Обычный |
| Юридическое | Договоры, NDA, ИП/СЗ оформление, комплаенс | Высокий — даже 1 задача может быть критичной |
| Финансы | Учёт предоплат, комиссии амбассадоров, бюджет, транзакции | Высокий — даже 1 задача может быть критичной |

## Маппинг старых названий сфер
- `продукт` → Амбассадоры
- `операции` → IT
- `исследования` → Методология

## Логика анализа

1. Посчитай количество задач в каждой сфере
2. Определи перекосы:
   - Если одна сфера > 40% задач — фокус слишком узкий
   - Если сфера с высоким весом (Юридическое, Финансы) пустая — это критично
   - Если 3+ сфер пустые — проект движется однобоко
3. Учитывай: юридических и финансовых задач может быть мало по количеству, но по значимости они могут быть выше чем 10 маркетинговых

## Формат ответа

Напиши 2-3 предложения в `team/core/sphere-advice.md`. Без заголовков, без markdown — чистый текст. Стиль: конкретный, деловой, без воды. Пример:

> Все задачи сосредоточены в Маркетинге (60%) и Партнёрствах (40%). IT и Методология пустые — без них платформа не развивается. Юридическое критично: нет задач по договорам амбассадоров, хотя уже 17 человек с нейробендами.

## Важно
- Не хвали, подсвечивай просадки
- Фокус на том, что НУЖНО ДЕЛАТЬ, а не на том что уже сделано
- Учитывай вес сферы: 0 задач в Юридическом опаснее чем 0 в Маркетинге
