# Промт: Автоклассификация задачи по сфере проекта

## Когда использовать
Каждый раз когда участник ставит задачу через бота (кнопка ЗАДАЧА или команда team_core_task).

## 7 сфер проекта

| Сфера | Ключ в JSON | Что входит | Примеры задач |
|-------|-------------|-----------|---------------|
| Амбассадоры | `амбассадоры` | Работа с амбассадорами, продажи нейробендов, реферальная сеть, обучение, ивенты для амбассадоров, промокоды | «Провести онбординг нового амбассадора», «Обновить условия промо» |
| IT | `IT` | Разработка бота, сайт, вёрстка, API, автоматизация, код, баги, деплой, интеграции | «Починить баг в боте», «Добавить кнопку выгрузки CSV» |
| Методология | `методология` | Протоколы MNPI, дизайн исследования, PI-индекс, научная часть, ЭЭГ-анализ, Neiry-сессии | «Описать протокол 6 сессий», «Разработать формулу PI» |
| Партнёрства | `партнёрства` | Клиники, внешние партнёры, Neiry, LQ Lab, презентации для партнёров, переговоры | «Подготовить презентацию для клиник», «Связаться с Neiry» |
| Маркетинг | `маркетинг` | Контент, статьи, блог, соцсети, бренд, видео, дизайн, Canva | «Написать статью для блога», «Сделать пост в Instagram» |
| Юридическое | `юридическое` | Договоры, NDA, оформление ИП/СЗ, комплаенс, лицензии, персональные данные | «Подготовить договор амбассадора», «Обновить NDA» |
| Финансы | `финансы` | Бюджет, учёт предоплат, комиссии, расчёты с амбассадорами, счета, налоги | «Свести учёт предоплат», «Рассчитать комиссии за апрель» |

## Правила классификации

1. Определи сферу по СОДЕРЖАНИЮ задачи, не по автору
2. Если задача касается нескольких сфер — выбирай ОСНОВНУЮ (на что направлено действие)
3. Примеры неочевидных:
   - «Сделать лендинг для амбассадоров» → IT (действие = вёрстка)
   - «Написать текст для лендинга» → Маркетинг (действие = контент)
   - «Подписать договор с клиникой» → Юридическое (действие = договор)
   - «Провести встречу с партнёром» → Партнёрства
   - «Исследовать Whoop API» → IT (действие = интеграция)
   - «Описать методологию для сайта» → Методология

## Формат ответа

Верни ТОЛЬКО ключ сферы из таблицы: `амбассадоры`, `IT`, `методология`, `партнёрства`, `маркетинг`, `юридическое`, `финансы`

Без объяснений, без кавычек — просто ключ.
