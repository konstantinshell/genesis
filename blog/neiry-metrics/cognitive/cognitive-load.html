Cognitive Load

ЧТО ЭТО

Интенсивность использования рабочей памяти и когнитивных ресурсов в данный момент. Следует перевёрнутой U-кривой Йеркса-Додсона: есть оптимальный уровень нагрузки, при котором эффективность максимальна.

КТО ОПИСАЛ

Роберт Йеркс и Джон Додсон в 1908 году описали связь между возбуждением и продуктивностью. Их закон применяется здесь: слишком мало нагрузки = скука, слишком много = перегрузка, оптимум посередине.

ЧТО ИЗМЕРЯЕТ

Сколько когнитивных ресурсов задействовано — от недогрузки (скука) до перегрузки (коллапс).

ДИАПАЗОНЫ
< 30 30–50 50–70 70–80 > 80
  • < 30 — скука, недогрузка, мозг «бездействует»
  • 30–50 — лёгкая нагрузка, рутинные задачи
  • 50–70 — оптимум — зона максимальной продуктивности
  • 70–80 — высокая нагрузка, работает, но на пределе
  • > 80 — перегрузка, эффективность падает
КРИТИЧЕСКИЕ КОМБИНАЦИИ

- Cognitive Load > 80 + Cognitive Control < 30 = когнитивный коллапс — мозг перегружен и не может регулировать себя. Необходим перерыв.

  • Cognitive Load 50–65 + Cognitive Control 65–80 = зона мастерства — оптимальная нагрузка при хорошей регуляции.
ЗАКРЫТЫЕ ГЛАЗА

Cognitive Load минимален (< 20). Мозг не обрабатывает задачу.

ОТКРЫТЫЕ ГЛАЗА, ФОКУС

Cognitive Load растёт пропорционально сложности задачи. Оптимум — 50–70.
