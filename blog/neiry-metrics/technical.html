Этот раздел предназначен для технических специалистов, владельцев устройств Neiry и тех, кто работает непосредственно в капсуле. Эти метрики не отражают состояние человека — они отражают состояние оборудования и качество данных.

---

IsConnected — статус подключения

Значения: true / false. Показывает, установлено ли Bluetooth-соединение с трекером.

Battery — заряд батареи

При заряде > 15% — качество записи нормальное. При < 15% — качество сигнала может снижаться, рекомендуется зарядить устройство.

Serial — серийный номер

6-значный числовой код, уникальный идентификатор устройства.

Firmware — версия прошивки

Числовой код версии. Обновления прошивки могут влиять на алгоритмы обработки.

Mode — режим работы

Текущий режим устройства (например, SignalResist).

Session Start — время начала сессии

Формат чч:мм:сс. Фиксирует момент начала записи.

Session Duration — длительность записи

Минимальные требования: > 5 минут для надёжного HRV, > 10 минут для оценки накопленной усталости.

Session ID — уникальный ID сессии

Буквенно-числовой код для идентификации конкретной записи.

Data Markup (1–5) — маркировка событий

Ручные метки, которые оператор ставит во время записи для обозначения значимых моментов (начало задачи, эмоциональное событие и т.д.).

Импеданс электродов (O1, T3, O2, T4)

  • Сопротивление контакта каждого электрода с кожей в кОм. Норма для сухих электродов: 100–300 кОм. Выше 300 — плохой контакт, нужно поправить электрод.

    Расположение электродов:

  • O1-T3 — отведение левого полушария (аналитическое мышление, речь)
  • O2-T4 — отведение правого полушария (пространственное восприятие, эмоции)

    Межполушарная асимметрия: при успешной деактивации DMN асимметрия обычно снижается — оба полушария синхронизируются. Это маркер успешной практики остановки внутреннего диалога.

Artifacts — наличие помех

No = чистый сигнал. Yes = обнаружены артефакты (мышечные, от движения и др.). При артефактах данные ЭЭГ ненадёжны.

Average µV — средняя амплитуда ЭЭГ

Средняя амплитуда сигнала в микровольтах. Норма: 10–100 µV. Значения 5–6 µV — ниже нормы, возможны проблемы с контактом.

PPG quality avg — качество PPG-сигнала

Интегральная оценка качества оптического сигнала. < 0.5 — ненадёжно, > 0.7 — приемлемо.

Is Calibrated — статус калибровки

Yes / No. Без калибровки SAT и персонализированные метрики (Emotions) недостоверны.
