Итоговые статусы

СОДЕРЖАНИЕ
--- Stress Recommendation State

---

Stress

ЧТО ЭТО

Бинарная итоговая оценка: «No stress» или «Stress detected». Формируется по совокупности PPG + ЭЭГ данных.

ИНТЕРПРЕТАЦИЯ

Это упрощённый индикатор для быстрого понимания. Для детального анализа используйте SI и SAT.

Recommendation

ЧТО ЭТО

Системная рекомендация по текущему состоянию — может предложить перерыв, дыхательное упражнение, смену активности или отсутствие рекомендации.

State

ЧТО ЭТО

Valid State / Invalid State — абсолютный приоритет среди всех метрик. Показывает, можно ли доверять остальным данным.

INVALID STATE

Данные ненадёжны — возможны проблемы с контактом электродов, артефакты, плохой PPG-сигнал. При Invalid State все остальные метрики не интерпретируются.

ЧТО ДЕЛАТЬ

Проверить электроды, убедиться в контакте с кожей, уменьшить движения, проверить перфузию.
