Производные индексы

Производные от спектральных мощностей ритмов. Персонализируются через baseline после калибровки.

Relaxation Index

Alpha / Beta

Простое соотношение Alpha / Beta.

< 0.5 0.5–1.5 > 2.0

Concentration Index

Beta / Alpha

Обратная величина Relaxation Index: Beta / Alpha.

< 1.0 1.0–2.0 2.0–3.5 3.5–6.0

Fatigue Score

(Alpha + Theta) / Beta

Моментальный уровень когнитивной усталости.

0.3–0.8 0.8–1.0 1.0–2.0 2.0–3.0

Reverse Fatigue

Beta / (Alpha + Theta)

Обратный индекс усталости — уровень бодрости.

> 2.0 1.0–2.0 < 1.0

Accumulated Fatigue

Накопленная усталость за сессию — интегральный показатель, который учитывает не моментальное состояние, а динамику усталости за весь период записи.

0–3 3–6 6–8 > 8

Productivity Score

Комплексная когнитивная эффективность — составной балл, учитывающий внимание, утомление, когнитивный контроль и нагрузку.

< 40% 40–60% 60–80% > 80%

Alpha Gravity

A2 / A1

Соотношение верхней альфы (A2, выше IAPF) к нижней альфе (A1, ниже IAPF).

< 0.8 ~1.0 > 1.2
