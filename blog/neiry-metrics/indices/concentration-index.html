Concentration Index

Beta / Alpha
ЧТО ЭТО

Обратная величина Relaxation Index: Beta / Alpha. Показывает степень когнитивной активации относительно расслабления.

ДИАПАЗОНЫ
< 1.0 1.0–2.0 2.0–3.5 3.5–6.0 > 6.0
  • < 1.0 — рассеянность, преобладание альфы
  • 1.0–2.0 — умеренная концентрация
  • 2.0–3.5 — хорошая концентрация
  • 3.5–6.0 — высокая концентрация
  • > 6.0 — перенапряжение, когнитивная перегрузка
МИФЫ

- «Чем выше Concentration Index, тем лучше». Нет. Выше 6.0 — это перенапряжение, эффективность начинает падать.

ЗАКРЫТЫЕ ГЛАЗА

Concentration Index низкий (< 1.0) — бета минимальна, альфа доминирует.

ОТКРЫТЫЕ ГЛАЗА, ФОКУС

Concentration Index растёт до 2.0–4.0 при нормальной работе.
