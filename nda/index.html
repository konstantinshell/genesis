1. Предмет соглашения

Настоящее Соглашение регулирует условия участия в исследовательской программе ONTO NOTHING, включая сбор, обработку и хранение персональных данных, данных нейроинтерфейсов, фитнес-трекеров и других источников биометрической информации.

2. Политика конфиденциальности

2.1 Какие данные мы собираем

  • Персональные данные (имя, контакты, город, дата рождения)
  • Данные анкеты (цели, методы, оборудование)
  • Данные нейротрекера NEIRY (ЭЭГ, концентрация, релаксация, стресс)
  • Данные фитнес-трекеров (пульс, HRV, сон, активность)
  • Клинические анализы (по вашему согласию)
  • Рефлексии и текстовые записи
  • Аудио-транскрипции сессий

2.2 Как мы используем данные

  • Для расчёта индексов MNPI и персонализации программы
  • Для научного анализа и публикаций (в анонимизированном виде)
  • Для генерации персональных отчётов и рекомендаций

Мы НЕ передаём ваши данные третьим лицам. Мы НЕ продаём и НЕ распространяем персональную информацию. Ваши данные используются исключительно в рамках исследовательской программы.

2.3 Хранение данных

Данные хранятся на защищённых серверах с шифрованием. Доступ имеют только авторизованные исследователи программы. Вы можете запросить удаление своих данных в любой момент.

3. Соглашение о неразглашении (NDA)

3.1 Обязательства RHYTHMS NOTHING

  • Не разглашать персональные данные участника
  • Не передавать индивидуальные результаты третьим лицам без согласия
  • Обеспечить безопасное хранение всех данных
  • Уведомлять участника об изменениях в условиях

3.2 Обязательства Участника

  • Не распространять методологию исследования (PI/MNPI) без согласия
  • Не копировать и не передавать материалы программы
  • Не публиковать результаты исследования от имени программы
  • Сохранять конфиденциальность данных других участников

3.3 Интеллектуальная собственность

Методология PI/MNPI, аудио-сессии, протоколы исследования и алгоритмы анализа являются интеллектуальной собственностью RHYTHMS NOTHING. Участник получает право использовать результаты для личного развития.

4. Права участника

  • Доступ к своим персональным данным в любой момент
  • Запрос на удаление данных и выход из программы
  • Получение персональных отчётов и рекомендаций
  • Участие в амбассадорской программе (по желанию)
  • Отзыв согласия на обработку данных

5. Срок действия

Соглашение вступает в силу с момента принятия (нажатия кнопки подтверждения) и действует на весь период участия в программе. При выходе из программы обязательства по конфиденциальности сохраняются в течение 3 (трёх) лет.
