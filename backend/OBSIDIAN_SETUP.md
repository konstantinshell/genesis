# Интеграция с Obsidian

Когда бэкенд создаёт файлы, они автоматически появляются в твоей папке Obsidian.

## Как добавить папку в Obsidian

1. Открой Obsidian
2. Создай новое хранилище (Vault) или открой существующее
3. **File → Open folder as vault** → выбери `/Users/konstantin/Documents/Obsidian Vault`
4. Готово! Теперь в левой панели будет папка "Данные участников"

## Структура файлов

После первой загрузки данных появится структура:

```
Obsidian Vault/
└── Данные участников/
    ├── Константин Шель/
    │   ├── Session_1.md
    │   ├── Session_1_metrics.csv
    │   ├── Session_2.md
    │   └── Session_2_metrics.csv
    └── Другой Участник/
        ├── Session_1.md
        └── Session_1_metrics.csv
```

## Рекомендуемые плагины Obsidian

Вот плагины которые сделают работу удобнее:

### 1. **Dataview** (бесплатный)
Позволяет создавать умные таблицы со всеми сессиями.

После установки создай файл `/Данные участников/Overview.md`:

```markdown
# Обзор всех сессий

```dataview
TABLE session, phone, file.mtime as "Последнее обновление"
FROM "Данные участников"
WHERE file.name != "Overview"
SORT file.mtime DESC
```

# Сессии по участникам

```dataview
TABLE sessions, file.name as "Участник"
FROM "Данные участников"
GROUP BY file.folder
```
```

### 2. **Obsidian Templater** (бесплатный)
Позволяет создавать шаблоны для новых сессий (опционально).

### 3. **CSV Editor** (бесплатный)
Красиво отображает CSV файлы таблицей в Obsidian.

## Синхронизация с облаком

Если хочешь доступ к данным с нескольких устройств:

### Вариант 1: Obsidian Sync (официальный)
- $96/год
- Встроенная синхронизация между устройствами
- Шифрование
- История версий

### Вариант 2: iCloud Drive (бесплатно)
1. Перемести папку Obsidian в `/Users/konstantin/Library/CloudStorage/iCloud~/Documents/Obsidian Vault`
2. Откройся эту папку в Obsidian
3. Автоматическая синхронизация на всех устройствах

### Вариант 3: Google Drive (бесплатно, до 15GB)
1. Установи Google Drive for Desktop
2. Включи синхронизацию для папки Obsidian Vault
3. Готово!

### Вариант 4: OneDrive (бесплатно, до 5GB + Office 365)
Похоже на Google Drive — просто синхронизируй папку.

## Резервная копия

Всегда делай резервные копии важных данных!

### Автоматическая резервная копия в Git

В папке Obsidian Vault:
```bash
git init
git add .
git commit -m "Backup of research data"
```

Можно запустить это через LaunchAgent каждый день.

## Проблемы

### Файлы не синхронизируются в Obsidian
- Перезагрузи Obsidian (Cmd+Q, потом снова открой)
- Проверь что папка правильная в File → Open vault settings

### CSV показывается как текст
- Установи плагин "CSV Editor"
- Или открой через текстовый редактор

### Слишком медленная синхронизация
- Если используешь облако, проверь скорость интернета
- Если папка очень большая, оптимизируй историю файлов

## Скрипт для автоматического коммита в Git

```bash
#!/bin/bash
cd "/Users/konstantin/Documents/Obsidian Vault"
git add -A
git commit -m "Backup $(date '+%Y-%m-%d %H:%M:%S')" || true
git push origin main || true
```

Сохрани как `backup.sh` и добавь в LaunchAgent чтобы запускалось ежедневно.
